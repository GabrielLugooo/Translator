### Objective

This project aims to develop a simple translator in Python that allows users to translate text from Spanish to English. It uses the `translate` library to facilitate the automatic conversion of phrases and words.

The goal is to learn how to integrate automatic translation tools in Python and provide a simple and fast solution for text conversion between languages.

### Skills Learned

- Use of the `translate` library in Python.
- Capturing user input with `input()`.
- Handling text strings in Python.
- Implementing automated language conversion.

### Project

#### Code with Comments (English)

```python
# Translator

# Install the necessary libraries
# pip install translate

# Import the necessary library
from translate import Translator # Module for automatic translation

# Create a Translator object to translate from Spanish to English
translator = Translator(from_lang='spanish', to_lang='english')

# Ask the user to enter the text to be translated
txt = input('What do you want to translate? ')

# Translate the entered text
res = translator.translate(txt)

# Print the translated result
print(res)
```
